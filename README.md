<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

First install Tkinter.
* Terminal
  ```sh
  pip install tk
  ```

After install Pillow.
* Terminal
  ```sh
  pip install pillow
  ```

## Run

```sh
python3 main.py
```
